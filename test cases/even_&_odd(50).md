# Test Cases: Count Even and Odd Numbers in Array

---

## Test Case 1: Mixed even and odd numbers
**Input:**
5
1
2
3
4
5

**Output:**
Number of even numbers: 2
Number of odd numbers: 3



## Test Case 2: All even numbers
**Input:**
4
2
4
6
8

**Output:**
Number of even numbers: 4
Number of odd numbers: 0



## Test Case 3: All odd numbers
**Input:**
3
1
3
5

**Output:**
Number of even numbers: 0
Number of odd numbers: 3



## Test Case 4: Single element, even
**Input:**
1
2

**Output:**
Number of even numbers: 1
Number of odd numbers: 0



## Test Case 5: Single element, odd
**Input:**
1
7

**Output:**
Number of even numbers: 0
Number of odd numbers: 1


## Test Case 6: Array size zero
**Input:**
0

**Output:**
Array size must be greater than 0.
