# Test Cases: Array Element Search

---

## Test Case 1: Element present in the array
**Input:**
5
apple
banana
cherry
date
fig
cherry

**Output:**
Element cherry found at position 3

## Test Case 2: Element not present in the array
**Input:**
4
1
2
3
4
5

**Output:**
Element 5 not found in the array



## Test Case 3: Single-element array, element found
**Input:**
1
hello
hello

**Output:**
Element hello found at position 1



## Test Case 4: Single-element array, element not found
**Input:**
1
world
hello


**Output:**
Element hello not found in the array


## Test Case 5: Array with duplicate elements
**Input:**
6
a
b
c
b
d
e
b

**Output:**
Element b found at position 2
