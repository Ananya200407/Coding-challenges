# Test Cases: Array Input and Display

---

## Test Case 1: Normal string elements
**Input:**
4
apple
banana
cherry
date

**Output:**
Array elements: ['apple', 'banana', 'cherry', 'date']


## Test Case 2: Numeric string elements
**Input:**
5
1
2
3
4
5

**Output:**
Array elements: ['1', '2', '3', '4', '5']


## Test Case 3: Single element
**Input:**
1
hello

**Output:**
Array elements: ['hello']


## Test Case 4: Mixed strings and numbers
**Input:**
3
apple
10
banana

**Output:**
Array elements: ['apple', '10', 'banana']

## Test Case 5: Empty strings
**Input:**
3

hello

**Output:**
Array elements: ['', ' ', 'hello']

## Test Case 6: Array size zero
**Input:**
0

**Output:**
Array size must be greater than 0.
