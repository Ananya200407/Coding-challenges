# Test Cases for Series of Squares Excluding 4 up to N

---

## Test Case 1: N = 16
**Input:**
- N: 16

**Expected Output:**
- Series up to 16 is:
- 1, 9, 16

---

## Test Case 2: N = 10
**Input:**
- N: 10

**Expected Output:**
- Series up to 10 is:
- 1, 9

---

## Test Case 3: N = 4

- N: 4

**Expected Output:**
- Series up to 4 is:
- 1

## Test Case 5: N = 1

## Test Case 6: N = 0

## Test Case 7: Negative input
**Input:**
- N: -5

**Expected Output:**
- No terms in the series for N < -5

---

## Test Case 8: Non-numeric input (alphabetic)
**Input:**
- N: abc

**Expected Output:**
- Invalid input! Please enter a positive integer.

---

## Test Case 9: Non-numeric input (special characters)
**Input:**

**Expected Output:**

**Expected Output:**
